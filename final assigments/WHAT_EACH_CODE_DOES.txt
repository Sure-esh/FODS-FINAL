WEEK 2 - QUESTION 3: ARMSTRONG NUMBER CHECKER
================================================================================
WHAT THIS CODE DOES:
This program checks if a number is an ARMSTRONG NUMBER.

Armstrong Number = Sum of digits raised to power of digit count equals the number
Example: 153 = 1³ + 5³ + 3³ = 1 + 125 + 27 = 153 ✓

PURPOSE:
- Learn string to number conversion
- Learn list comprehension
- Learn mathematical validation
- Understand special numbers

HOW IT WORKS:
1. Take number as input
2. Count digits
3. Raise each digit to power of digit count
4. Sum all these powers
5. Check if sum equals original number
6. Show result

EXAMPLE:
Input: 153
Check: 1³ + 5³ + 3³ = 1 + 125 + 27 = 153
Output: YES, it's Armstrong

Input: 123
Check: 1³ + 2³ + 3³ = 1 + 8 + 27 = 36 ≠ 123
Output: NO, it's not Armstrong

REAL-WORLD USE:
Number theory, mathematics education, recreational programming
